# inventario-BI
proyecto de inventario calzado (data_shoe)

Proyecto de análisis de inventario desarrollado con:

- SQL (base de datos de productos)
- Power BI (dashboard)
- GitHub (control de versiones)

## Funcionalidades

- Análisis de stock por marca
- Valor total del inventario
- Total de pares disponibles
- Visualización de datos mediante dashboard

## Tecnologías utilizadas

- SQL
- Power BI
- Visual Studio Code
